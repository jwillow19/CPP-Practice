    + `!, &&, ||`
+ User input
    + `cin, cin.get(), getline`

### Control Statements
3 main types of control statements: **sequential, selection, repetitive**

+ Sequential - default programming (line-by-line execution)
+ Selection - decision statements
    + `if, else-if, else, switch (case, default)`
+ Repetition - loops
    + `for, while, do-while`
